    How to play?

  1. Width and Height are required parameters
  2. Max width and height value is 20, just because it's the most comfortable size for your eyes
  3. Percentage is optional. If You don't enter it, game starts with a random value from 1% to 100%
  4. There is a checker above the game field. It shows if game can be solved or not.
  5. If game can be solved, you will see a number of moves to solve a puzzle(fastest option)
  6. If game can't be solved, please press 'Restart' button
  7. Restart button will set a new field with the same parameters
  8. If you want to set new parameters, click 'Main Menu' button
  9. Main menu will move you to a first screen(by refreshing page, be carefull with your web connection
  10. Hard mode covers 20% of field with additional holes. Be careful, it may shut down all routes to a hat
  11. You can move your character using either arrows on the screen or arrow keys on your keyboard
  12. If you want to get in touch with creator, click the link below
  13. Thanks for your attention! Good luck and Have fun!
